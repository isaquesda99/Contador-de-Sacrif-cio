# **App Name**: Study Time Tracker

## Core Features:

- Study Session Timer: Start, pause, reset, and save study session times.
- Time Saving: Saves study sessions in the Firestore database for persistence and retrieval.
- Daily Study History: Displays a log of study times for each day.
- Weekly Study Summary: Calculates and shows the total study time for the current week.
- Monthly Study Summary: Aggregates and displays the total study time for the current month.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to promote focus.
- Background color: Light gray (#FAFAFA) for a clean and distraction-free interface.
- Accent color: Soft green (#A5D6A7) for start/save buttons to indicate progress.
- Body and headline font: 'Inter', a modern sans-serif, for a clean and readable UI.
- Simple, outlined icons for timer controls (start, pause, reset, save).
- Clean, card-based layout to separate the timer, daily history, and weekly/monthly summaries.
- Subtle progress animations to visualize the time elapsed during a study session.
